# Clery Data Dictionary Changelog

## [1.0.0] - 2026-03-09

### Added

- **clery_crime_definitions.json**: Clery Act crime categories with NIBRS mappings
  - Primary crimes (11 categories)
  - Hate crime additions (4 categories)
  - Arrest/referral categories (3)
  - VAWA crimes (3)
  - Bias categories

- **clery_geography_definitions.json**: Clery geographic categories
  - On-Campus, Noncampus, Public Property
  - Definitions per DOE Handbook 2016

- **nibrs_to_clery_map.json**: NIBRS → Clery mapping
  - 25+ NIBRS code mappings
  - Confidence levels and hate-crime-only flags
  - VAWA classification notes

- **REQUEST_CHECKLIST.md**: Step-by-step checklist for completing Clery requests
  - Data scope, crime categories, geography filtering
  - Classification steps, quality checks
  - Reference to institution-specific docs (e.g., FDU PDF)

- **README.md**: Overview and quick start
